🔄 Studio to Module Converter

Transform your Odoo Studio customizations into version-controlled module code!

✨ Key Features

📋 List Studio Views
Browse and manage all views created through Odoo Studio in one place.
🔄 Batch Conversion
Convert multiple Studio views to module code at once.
📝 Smart XML Generation
Automatically generates proper, well-formatted XML view files.
🔧 Module Integration
Automatically updates your module's __manifest__.py file.
🗑️ Auto Cleanup
Optionally removes Studio views after successful module upgrade.
👁️ Preview
See the generated XML before committing to conversion.

🔄 Workflow

1. Create

Build customizations in Studio

2. Convert

Use this tool to generate XML

3. Review

Check generated files

4. Deploy

Upgrade module

🚀 Quick Start

  1. Go to Studio to Module > Studio Views
  2. Select the views you want to convert
  3. Click Action > Convert Studio Views
  4. Choose your target module
  5. Click Convert Views
  6. Upgrade the target module
⚠️ Important: Always test conversions on a development server first. Make sure you have write access to the target module directory and that the module is in your addons path.

💡 Use Cases

  • Development Workflow: Prototype in Studio, deploy as code
  • Version Control: Track view changes in Git
  • Multi-Environment: Easily deploy customizations across servers
  • Code Review: Review customizations as code before production
  • Backup: Keep customizations safe in version control

For detailed documentation, see the README.md file in the module directory.
